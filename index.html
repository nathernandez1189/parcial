¿Cómo configurar un archivo .gitignore?

El archivo .gitignore es utilizado para indicar a Git qué archivos y carpetas deben ser ignorados y no rastreados en el repositorio. A continuación, pondre su respectivo paso a paso:

Paso 1: Crear el archivo .gitignore

En la raíz de tu proyecto, crea un archivo llamado .gitignore. Se puede hacer manualmente o utilizando un comando en la terminal:

touch .gitignore

Paso 2: Editar el archivo .gitignore

Abre el archivo .gitignore en el editor de texto y agrega las rutas de los archivos y carpetas que se desea que Git ignore.

# Ignorar la carpeta node_modules
    node_modules/
    
    # Ignorar archivos de entorno
    .env
    
    # Ignorar archivos compilados
    dist/
    build/

Paso 3: Guardar y confirmar los cambios

Se guarda los cambios en el archivo .gitignore y realiza un commit para aplicar la configuración:

git add .gitignore
    git commit -m "Añadir configuración de .gitignore"
    git push origin main
